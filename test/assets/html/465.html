What is Cloud Computing?

Tuesday March 24, 2009

We've touched briefly on the concept of cloud computing and cloud computing companies in the past, but it's a topic that definitely deserves a more in-depth look. Cloud computing is one of those concepts/models that the more you talk about it, the less you seem to understand what it's all about. It's hard to grasp, much like a cloud itself, in part because what cloud computing means to one person is very different than what it means to the next. Then, just when you think you've got the shape of it, it morphs into something else and you have more questions.

There are a number of very intelligent, talented people saying "I just don't understand the cloud computing model" or "I understand this concept, but how is it affecting me, my family or my business?" Of course, for Ruby programmers the question takes a slightly different tack: How does Ruby fit into cloud computing? Back to Basics: What is Ruby Anyway?

Monday March 23, 2009

It's all fine and good to give you an overview of how to create regular expressions, use RubyGems or play with some interesting scripts, but the bottom line is if you don't know the basics of what makes Ruby Ruby, then none of the rest is of any use.

So what is Ruby anyway? Well, according to Yukihiro "Matz" Matsumoto, Ruby's creator, it's a fun programming language. We've all heard the buzz word "user-friendly," but Ruby goes beyond user-friendly. It's a language that is human-oriented instead machine-oriented. It's pure object-orientation is designed to reduce confusion instead of making you paw through lines of code to find the "gem" inside.

So if you haven't taken the time to really get to know your programming language, what it can do (and what it can't), and how it can be used, today might be the day to discover all about Ruby.

This Week's New Releases

Friday March 20, 2009

There's one release this week that simply cannot be ignored: Rails 2.3. One of the big improvements here is the Rack integration. Rack is a kind of middleware that sits between your Rails application and the web server. This means Rails apps can run on any server Rack supports. See our announcement and the official announcement for more information.

But more has been released this week than a new version of Rails.

  • Hpricot 0.7

    There's a battle heating up between Hpricot and the newcomers: Nokogiri and LibXML. Hpricot is smaller and simpler than both of them, and suitable for parsing HTML as well. This new version improves performance further.

  • JRuby 1.2.0

    We reported last week on the JRuby 1.2.0 release candidate. Well, the final release is here, and is ready for widespread use. Highlights are better Ruby 1.9 support and performance improvements in the parser (sometimes up to 6 times faster).

  • RSpec 1.2.0

    RSpec is a Behavior Driven Development tool for Ruby. With release 1.2.0 come a few minor changes. If you've been running the ruby command on your test files, your tests will no longer run. Use the spec command instead. Also, the setup and teardown methods are gone, use before and after instead. There are also a number of small improvements and bug fixes.
