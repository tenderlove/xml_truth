Ruby on Rails Plugins

Welcome to the Rails Plugin Directory, where you can find all Rails plugins you could possibly want. I'm always on the hunt for new plugins to add to the directory, so you can be lazy and just come here to find the latest and greatest contributions from the Rails community. There are RSS feeds to keep you up to date, and a recently added category if you're into the old-school way of getting updates.

If you'd like to learn more about what plugins are and how they fit into the Ruby on Rails framework, Geoffrey Grosenbach has some excellent tips regarding plugins in the Complete Guide to Rails Plugins parts one and two. Of course, the plugins page from the Rails wiki also has some great information.

From here you can list all plugins, use the navigation at right to list plugins by category, or add your own.

Plugin Directory Accessories

Here are a few tools that interact with the plugin directory that should make it easier for you to find and install plugins.

  • Rapt: A command-line tool that replaces the plugin script that comes with the Rails distribution. You can search via the command line, thus never having to use 'discover' again. Available at http://rapt.rubyforge.org or via sudo gem install rapt.
  • The Rails bundle in TextMate maintained by Duane Johnson allows you to search for and install plugins from within TextMate.
  • A Firefox search box enhancement by Eric Anderson allows you to quickly complete a plugin search here at the site.
