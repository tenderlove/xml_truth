Ruby Multimedia Icons

Ruby Multimedia icon set is the new creation of DryIcons' laboratory. Using only two colors, red and white, we brought style and class to the next level. This set can find its use in lots of projects and applications. It contains 13 high quality web icons in 32-bit transparency PNG, ICO and ICNS formats. They come in 16 x 16px; 24 x 24px; 32 x 32px; 48 x 48px and 128 x 128px size.
