Ruby Projects


  • acoc, a program for colouring the output of any program.
  • nldict, a Dutch dictionary program.
  • rit, the Ruby IMAP tool.
  • Ruby/Amazon, a Ruby interface to v3 of the Amazon Web Services API. This library has been succeeded by Ruby/AWS.
  • Ruby/AWS, a Ruby interface to v4 of the Amazon Web Services API. This library is the successor to Ruby/Amazon.
  • Ruby/CorporateTime, a Ruby interface to Oracle's Calendar application.
  • Ruby/DICT, a DICT protocol library and client
  • Ruby/Finance, access to financial information, including currency conversion and stock quotes.
  • Ruby/Google, a Ruby interface to the Google Web API.
  • Ruby/LDAP, the popular Ruby library for interacting with LDAP servers. There is now extensive RDoc documentation available for the project.
  • Ruby/Password, a Ruby library for working with passwords.
  • Ruby/XMLTV, a TV programme schedule grabber for UPC's television network in The Netherlands.
  • xmms2pl, a program for connecting to XMMS and exporting playlists and their files. I use this for syncing with my iRiver iHP-140 multi-codec jukebox.
  • xmmsgrep, a program for connecting to XMMS and looking for regular expressions in song titles and artist names.
