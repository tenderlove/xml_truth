Ruby Code Samples for the Live Search API

Brief Description
Example code for calling the Live Search SOAP web service in your Ruby application.

Overview

This code sample demonstrates how to use the Live Search SOAP web service in your Ruby application. It includes examples of all major functionality provided by the webservice and include a reusable module to simplify the API calls.

For the latest updates and news on the Live Search API, please subscribe to the Live Search Developer's Blog.

System Requirements

  • Supported Operating Systems: Unix; Windows 2000; Windows 98; Windows Vista; Windows XP
Please refer to the README.html file in the download archive for detailed information about requirements. Following is a summary of the major requirements:

- Ruby 1.8.5 (http://www.ruby-lang.org/en/downloads)
- SOAP4R (http://dev.ctor.org/download/soap4r-20061022.zip)
- eRuby 1.0.5 (http://mirrors.ibiblio.org/pub/mirrors/ruby/binaries/mswin32/ext/eruby-1.0.5-i386-mswin32-1.8.zip)
- An Application ID that will allow this sample to access the Live Search Web Service
