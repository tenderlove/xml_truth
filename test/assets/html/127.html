NetBeans Ruby Support

The NetBeans Ruby support plug-ins provide an integrated development environment for building, running, testing, and debugging Ruby and Ruby on Rails applications. You can download a Ruby-only version of the NetBeans IDE or add Ruby support to your NetBeans IDE download. To learn how, see Installing and Configuring Ruby Support. If you want to work on the bleeding edge, see the Ruby Installation wiki page for information on getting the latest builds. The Ruby Recent Changes page tracks the newest features.
